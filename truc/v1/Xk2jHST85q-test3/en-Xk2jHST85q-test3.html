What is MadMapper?

MadMapper is a professional video mapping software developed by GarageCUBE that allows artists, designers, and technicians to project visual content onto complex surfaces and objects. It transforms any surface into a dynamic canvas for creative expression through precise geometric calibration and real-time visual effects.

Key Features and Capabilities

Surface Mapping

MadMapper excels at surface mapping, which involves projecting images and videos onto irregular surfaces like buildings, sculptures, or stage designs. The software calculates the exact geometry needed to make flat digital content appear perfectly fitted to three-dimensional objects.

Real-Time Performance

The software is designed for live performances and installations, offering low-latency processing that makes it suitable for concerts, theater productions, and interactive exhibitions. You can manipulate visuals in real-time without noticeable delays.

Multi-Projector Support

MadMapper supports multiple projectors simultaneously, allowing you to create large-scale installations that span across various surfaces. The software handles edge blending and geometric correction across multiple projection sources.

Technical Workflow

Surface Creation

The mapping process begins with surface creation, where you define the shape and position of your projection target. MadMapper provides tools to create quads, triangles, circles, and complex polygonal shapes that match your physical surfaces.

Content Assignment

Once surfaces are defined, you can assign various content types including:

  • Video files
  • Live camera feeds
  • Generated patterns
  • Audio-reactive visuals
  • Interactive elements

Calibration and Alignment

The software includes precise calibration tools to ensure your projected content aligns perfectly with physical surfaces. This involves adjusting corner points, applying perspective corrections, and fine-tuning geometric parameters.

Interface and User Experience

Intuitive Design

MadMapper features a user-friendly interface that balances professional capabilities with accessibility. The workspace is organized logically, making it easier for both beginners and experienced users to navigate complex projects.

Modular Architecture

The software employs a modular approach where different components can be combined and configured according to project needs. This flexibility allows for both simple single-surface mappings and complex multi-element installations.
