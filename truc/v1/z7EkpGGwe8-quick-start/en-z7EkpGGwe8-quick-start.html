Quick Start

General

MiniMad is an all-in-one dedicated media playback device for video and LED mapping, based on the Raspberry Pi platform. Easy to use, small and versatile, it can be a good replacement for a computer in your future audiovisual installation.

The MiniMad works with a SD-card that contains both the system and the software of the MiniMad. The MiniMad system can be updated or reinstalled if needed, by following the procedure described in the chapter called “Flash your MiniMad SD-card”. The MiniMad software is getting automatically updated when exporting from an updated version of MadMapper.

MiniMad can either be installed with MiniMad Video to play video-mapping, or with MiniMad Light to play recorded DMX sequences. Note that MiniMad Video and MiniMad Light cannot be installed at the same time on the same MiniMad.

Finally, the SD-card must be activated for each new MiniMad it is used into. The SD-card is already activated in any brand new MiniMad. If you want to use a SD-card other than the one the MiniMad shipped with, please make sure to flash the latest system by following the chapter called “Flash your MiniMad SD-card” prior to activate it. At export time, MadMapper will remind you to activate when trying to export to a SD-card that was not yet activated.

 

Turning the MiniMad On

Turning the MiniMad On is as simple as plugging it in. To avoid plugging it in and out, you can use an AC power-switch. If you want to schedule MiniMad’s playback during the day, please refer to the TIPS & TRICKS section.

Turning the MiniMad Off

Depending on the chosen export settings, you can safely disconnect the MiniMad from power, or be required to shutting it down properly to avoid data-corruption on the SD-card. Basically, data-corruptions happens when cutting the power of the MiniMad while it’s writing on the SD-card, which can happen:

  • when changing media, when the “start last played media” export setting is checked
  • when changing master luminosity
  • when changing audio volume
  • when changing playback mode
  • when editing using the mapping or image control modes in MiniMad Video

By consequence, we strongly advice against turning off the MiniMad by cutting the power if you use any of the following features. Instead, you should perform a proper shutdown using either the MiniMad buttons, a USB keyboard, MiniMad Controller or using OSC. Please refer to the CONTROLS section for more details.

Activate your MiniMad SD card

If you flashed your MiniMad SD-card, or slot-in an SD-card that was used in another MiniMad unit, you will need to activate it prior to being able to export to, or use in the MiniMad.

 

The activation procedure is quick though:

1. Insert the SD-card in the MiniMad

2. Connect the MiniMad to a HDMI display *

3. Plug the power cord to turn the MiniMad on 4. Follow the on-screen procedure

 

Note that the MiniMad red LED is blinking when the activation process is required, and stops blinking once the SD-card is activated.

 

(*) If you cannot connect to a HDMI display, wait for the red LED to start blinking, then successively press the red, yellow and blue buttons. If the operation was successful, the LED should stop blinking, meaning the SD-card is now activated, and can be export to.

Flash your MiniMad SD card

Flashing a SD-card with MiniMad system can be necessary in the following cases:

  • A newer version of the system is available, providing new features to the MiniMad. MadMapper will warn you if flashing is necessary.
  • You want to use a brand new SD-card in the MiniMad.
  • The system does not work anymore because of some data corruption. (cf. Turning off the Minimad)

You first need to download the latest image of the MiniMad system (8, 16GB and 32GB versions are available) as well as the Etcher flashing tool. All those files are available to download in MadMapper Downloads. Note that you don’t need to decompress the downloaded zipped images.

 

Then:

1. Insert your MiniMad SD-card in your computer’s card-reader.

2. Launch Etcher application.

3. Select the downloaded .zip disk-image file as source.

4. Select the SD-card drive as target.

5. Click “Flash!” to start the operation.

 

The flashing operation can take up to 20-30 minutes. The MiniMad image disk is big - if the image file doesn’t work it may be due to a failed download and you’ll have to download the image disk again.
