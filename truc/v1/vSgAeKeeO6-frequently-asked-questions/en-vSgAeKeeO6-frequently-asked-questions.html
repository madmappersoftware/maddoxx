MiniMad FAQ

Can I control the MiniMad from MadMapper ?

You can individually control any number of MiniMad connected to the network with the MiniMad Control Module (Modules Tab).

Can I control the MiniMad with a MIDI controller?

MiniMad supports the MicroModul8 MIDI controller only, in addition to buttons, usb/wireless keyboard, OSC and the MiniMad controller module. We might have plans for universal MIDI support for the MiniMad in the future, stay tuned.

Can I output to multiple projectors with the MiniMad?

The MiniMad has a single HDMI output so it can output to only one projector. Which means one MiniMad is required for each projector. Multiple MiniMad units will automatically synchronize themselves when connected to the same ethernet network.

Can I playback video and light at the same time from one MiniMad unit?

You currently can’t playback video AND light at the same time from one MiniMad, you would have to use different units.

Can I replace MadMapper with a MiniMad?

The MiniMad is essentially an advanced media player. So it is not intended to replace MadMapper and its more advanced features requiring the performances of a proper computer.

Can I use several SD cards with the MiniMad?

You can activate several SD cards for the same MiniMad and export different content on each one. It's then just a matter of changing the current card. Remark: If you buy your own SD-cards, make sure they're at least the same size as the MiniMad’s original one; make sure as well you don't get “cheap” ones. Also please note that each new SD-Card must be physically activated for each different MiniMad unit, pressing the Red/Yellow/Blue buttons successively when the LED is blinking in red.

Can I use soft-edge (edge blending) with the MiniMad?

The MiniMad doesn't do soft-edge / edge blending but it can play many video outputs in sync. Soft-edge requires MadMapper and a computer with all the video-outputs running from the same graphics-adapter (GPU) to ensure perfect time synchronization.

Can I use the MiniMad for my project?

MiniMad is a perfect companion to MadMapper, however running on a Raspberry Pi it can't totally replace it. We invite you to learn more about the device’s features through the website or the PDF guide.

Do I need MadMapper to use the MiniMad?

MadMapper is indeed required to export a video or a light mapping to the MiniMad’s SD card. Once exported, the MiniMad will playback your content autonomously.

Do I need to activate my purchased MiniMad?

When you buy a MiniMad, the unit is already activated. However, if for instance you change your SD card you will need to reactivate your MiniMad at startup with the buttons.

For how long can I run the MiniMad without rebooting it ?

The MiniMad, if running on an Industrial Grade SD card and well aired, should run a very long time. We’ve had units running for more than a year in suitable environments.

How many DMX channels are supported by the MiniMad?

You can playback up to 64 DMX universes with the MiniMad. Which corresponds to 32 768 channels.

How many MadMapper licenses do I need to use multiple MiniMad units?

You only need one MadMapper license to use as many MiniMad as you want. Exporting mapping projects to the MiniMad's SD card is unlimited.

How will the MiniMad behave after a power cut?

The MiniMad does support power cuts and failures, it will restart by itself once the power is back.
However, we advise to make a clone of your SD-card(s) in case a data corruption happens that could prevent the MiniMad from starting.

What are the technical specifications of the MiniMad?

Raspberry Pi 2, 7 buttons control panel, HDMI output up to 1080p (1920x1080 at 60Hz), size 9 x 6 x 3 cm (approx. 3.5 x 2.3 x 1.2 inches).

What is the largest video size file that can be played by the MiniMad?

There is no duration limit for your video files but they will not be able to exceed a size of 4GB (the maximal size of a file on a FAT32 partition) for a 16GB/32GB SD-card.

What is the recommend codec for MiniMad videos?

H264 and HAP are recommend. Please note that 1) MadMapper will take care of transcoding when exporting to MiniMad, 2) advanced settings are available in the Export window, 3) if you’re not satisfied with your results, re-export your project by loading default export settings.

Why would I need an Industrial Grade SD-card?

In some situations where space is limited, the MiniMad units might get hot. If too hot for a long time, data corruption may appear on entry level SD-card. In these situations we advise you to use an Industrial/Military grade SD card. Such cards will allow you to run your installation for a very long time.
